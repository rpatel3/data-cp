The Dataset¶

The NBA data set I used was scraped and put together using multiple different csv files. The website where I got my data, basketball-reference, had alot of statistics on each team throughout the 2015-2016 season. The question I had throughout this process was predicting which team would win an upcoming game based on their prior stats. I ran into a problem where I had to use seasonal averages rather than game-to-game data.

Exploring the dataset¶

Using pandas, I collected all the data together and cleaned the data from around 80 columns for each team to around 50 columns that were relevant to the team rather than their opponents.

Machine Learning¶

Features

The features I found useful after exploring more of the data were:

  • Efficient Field Goal Percentage
  • Turn overs per game
  • True Shooting percentage
  • Offensive Rating
  • Points per game
  • Defensive Rating
  • Number of wins

target = team2 beats team1

Using these features and the target, I used test_train_split to train my model and RandomForestClassifier to fit my model and create predictions.
